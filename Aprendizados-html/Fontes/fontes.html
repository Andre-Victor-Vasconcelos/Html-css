Vamos ver estilo de letras, fontes externas e do google

  • Fonte google

    • Só entrar no site google imagens, clicar em @import e atribuir a fonte

  • Fonte externa

    • Usar o @font-face no css (usei tag 'pre' na formatação do código): 

                        @font-face {
                    font-family: 'Book' ;
                    src: url('book_ends/Book+Ends.otf') format('opentype');
                }
        }

  • Fonte sublinhada

    • text-decoration: underline;

  • Fonte itálico

    • font-style: italic;

  • Fonte normal bold

    • font-weight: bold;

  • Fonte normal tudo maiúsculo

    • text-transform: uppercase;
